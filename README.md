# DeckDuel

DeckDuel is a small full-stack demo application built to showcase a typical modern web architecture.

The project consists of a .NET Minimal API backend and a React frontend client.

## Architecture

React Client
     │
     ▼
ASP.NET Core API -> Azure OpenAI
     │
     ▼
SQL Server

## Tech Stack

Backend
- ASP.NET Minimal API
- C#
- Entity Framework Core

Frontend
- React
- Chakra
- JavaScript
- HTML / CSS

## Project Structure

/api  
Contains the ASP.NET Core Web API solution.

/client  
Contains the React frontend application.
